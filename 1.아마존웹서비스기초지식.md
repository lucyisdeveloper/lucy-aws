# 1. 아마존 웹서비스 기초지식

### AWS란 ? (Amazon Web Service)
: 아마존이 제공하는 클라우드 컴퓨팅 서비스 중 하나
: 클라우드 컴퓨팅 서비스란, 서버및 네트워크 등을 인터넷으로 빌려주는 서비스 (언제, 어디서든 사용 OK)
→ 제공하는 서비스가 165개 이상으로 모든 Application과 Infrastructure 를 구축 OK
